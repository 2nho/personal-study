```
function arrayReverse(array){
	array.reverse();
	for(let i=0;i<array.length;i++) {
		if(Array.isArray(array[i])) {
			arrayReverse(array[i]);
		}
	}
}
```

- 이 함수를 이용해서 isArray를 통해 배열인지 확인 후 배열이면 뒤집기
